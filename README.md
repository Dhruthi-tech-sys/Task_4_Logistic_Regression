# Task 4: Logistic Regression

## Objective
Build a binary classifier using Logistic Regression.

## Dataset
Breast Cancer Wisconsin Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Steps
1. Loaded dataset
2. Cleaned data
3. Converted diagnosis labels
4. Train-test split
5. Standardized features
6. Trained Logistic Regression model
7. Evaluated using:
   - Confusion Matrix
   - Precision
   - Recall
   - F1 Score
   - ROC-AUC
8. Visualized ROC Curve
9. Visualized Sigmoid Function

## Result
Built a high-accuracy binary classification model for breast cancer prediction.
